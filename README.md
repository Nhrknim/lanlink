# LANLink

LANLink is a LAN-based real-time chat application built using Python and socket programming.

## Features Implemented

- Multi-client chat system
- Real-time message broadcasting
- Username support
- Join and leave notifications
- TCP client-server communication
- Multithreaded server for handling multiple users

## Technologies Used

- Python
- Socket Programming
- TCP/IP
- Threading

## How to Run

### Start the server

```bash
python server.py
```

### Start a client

```bash
python client.py
```

Enter the server IP address and a username when prompted.

## Current Status

The terminal-based chat system is complete. The next phase is integrating a PyQt6 graphical user interface and adding advanced features such as file sharing and encryption.
